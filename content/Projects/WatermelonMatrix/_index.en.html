{{% projectDescription %}} {{% projectDescriptionBox name="Description" %}} Watermelon Matrix is a C++ library containing implementations for vectors and matrices.
All functions are runnable at compile time, even complex ones such as the inverse function.

This library was made as a way to explore template mechanics.
It includes : - CRTP (Curiously Recurring Template Programming) - SFINAE (Substitution Failure is not an Error) - Template specialization - Explicit template instantiation {{% /projectDescriptionBox %}} {{% projectDescriptionBox name="Tools" %}} Programming: - C++ {{% /projectDescriptionBox %}} {{% /projectDescription %}}
