Discord Board Game

A board game made using a Discord Bot

{{% projectDescription %}} {{% projectDescriptionBox name="Description" %}} Still in development.
At first, I just got inspired to make a board game to play with friends, so I made semi automatic by making a discord bot using Discord.py to help with randomness, hiding cards, etc.
However, I then decided to let the bot render directly the game and handle all of the logic.
But what's better than a Game Engine to make a game? So I made a [wrapper of the Godot API](https://thomasprog.github.io/projects/discordwrapperforgodot) myself and used that to generate images. {{% /projectDescriptionBox %}} {{% projectDescriptionBox name="Tools" %}} Programming: - GDScript - Godot {{% /projectDescriptionBox %}} {{% /projectDescription %}} {{% projectDescription %}} {{% projectDescriptionBox name="My Work" %}} Programming: - Gameplay of a Board Game and Card Game {{% /projectDescriptionBox %}} {{% /projectDescription %}}
